# Mapa Mental: Estatística Salva-Vidas

Projeto estático e simples, feito só com `HTML`, `CSS` e `JavaScript`.

## Arquivos

- `index.html`: página principal com o mapa mental
- `styles.css`: layout e visual
- `script.js`: destaque de tópicos ao clicar

## Abrir localmente

Basta abrir `index.html` no navegador.

## Publicar no GitHub Pages

1. Crie um repositório no GitHub.
2. Envie a pasta `docs` com estes arquivos.
3. No GitHub, abra `Settings > Pages`.
4. Em `Build and deployment`, escolha `Deploy from a branch`.
5. Selecione a branch principal e a pasta `/docs`.
6. Salve e aguarde a URL do GitHub Pages.

Formato da URL:

`https://SEU-USUARIO.github.io/NOME-DO-REPO/`

## Observação

Não há build, dependência externa, React ou qualquer configuração adicional.
